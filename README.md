# yotaiyo-to-do-app-react-rails

### setup
```
git clone git@github.com:yotaiyo/yotaiyo-to-do-app-react-rails.git
cd yotaiyo-to-do-app-react-rails

- frontend
cd frontend
yarn
yarn test
yarn run dev
http://localhost:3000/ToDoScreen

- backend
cd backend
bin/rails s -p 3001
http://localhost:3001/todo
```
